- **Hero** — new tagline `Intelligent Environmental Control — Purpose-built for Tropical Asia`. Word-by-word reveal animation via GSAP. Sub copy emphasises 60% global cooling-demand growth from tropics. Hub 100 dashboard mockup (CO₂/PM2.5/RH/Temp + STATE NORMAL) floats over the existing Cloudflare Stream video on desktop (hidden ≤1024 px so mobile keeps cinematic video). - **Benefits** — `benefits-grid` becomes `bento-grid` (asymmetric 6-column). Card sizing: People = lg(4), Buildings = sm(2), Society = sm(2), Environment = lg(4) → 4+2 / 2+4. Mobile collapses to 2-col then 1-col.
- **Leadership** — every team-card now has a circular gold-gradient avatar slot with the person's initials (Cormorant italic), in front of name/role. Photo upload later just means swapping the slot div for an `<img>` tag; the CSS already supports `border-radius:50%`.

### Visual + interaction layer (new)

- **Lenis smooth scroll** (CDN, `unpkg/lenis@1.1.20`) — wired to GSAP's ticker so ScrollTrigger stays in sync. Anchor links use Lenis `scrollTo` with -80px offset for nav. Disabled if `prefers-reduced-motion`.
- **GSAP + ScrollTrigger** (CDN, `cdnjs 3.12.5`) — drives word-by-word hero reveal, tech-scroller stage toggling, and a `ScrollTrigger.refresh()` after layout settles.
- **Word-by-word reveal** (`.reveal-words`) — text walker preserves `<em>`, `<strong>`, `<br>`. Falls back to IntersectionObserver + CSS staggered transition-delay if libs fail to load.
- **Stagger reveal** (`.stagger-children`) — Patent pills, compliance badges, traction stats, traction logos all animate in with 70ms increments.
- **Custom cursor** — desktop only (`@media (hover:hover) and (pointer:fine)`). 6px gold dot tracks mouse 1:1; 34px ring lerps at 0.18 with mix-blend-mode difference. Ring grows to 54px over interactive elements (a/button/cards/badges/pills).
- **Number counters** (`[data-counter]`) — IntersectionObserver-driven, ease-out cubic, formatted with `toLocaleString('en-US')`. Existing hero counters kept; traction stats added.
- **Nav scroll-spy** — adds `.active-section` class to the nav link whose section is in the middle 5% of viewport.

### Brand consistency (kept)

- **TRUE BLACK** (`#000000`) bg untouched. Champagne gold `#C9A96E` accent unchanged.
- **Fonts** — Plus Jakarta Sans / Cormorant Garamond / Space Grotesk / Be Vietnam Pro. NO Inter/Geist swap (rejected — would break luxury identity).
- **Cloudflare Stream videos** — every existing video iframe untouched. Video UIDs unchanged.

## CDN dependencies added (4 lines)

```html
<script src="https://unpkg.com/lenis@1.1.20/dist/lenis.min.js" defer></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js" defer></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/ScrollTrigger.min.js" defer></script>
```

(Motion library `motion.dev` planned but skipped — GSAP already covers everything needed for v1.)

All loaded with `defer`. `whenLibsReady()` polls for them up to ~5 s; if any fail, the site degrades gracefully to CSS-only animations. No layout shift on slow networks.
